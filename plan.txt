최소한의 코드를 사용하자.


==== current situation ====

- Sponza scene loads and renders.
- Diffuse texture loading is working.
- Camera interaction is improved:
  - W/A/S/D move
  - Space / Left Shift move vertically
  - mouse look works
  - scroll zoom works
  - Esc releases mouse focus
  - left click captures mouse focus again
- Render path is cleaned compared with the early prototype.
- Deferred rendering migration has started.

현재 deferred 상태
- G-buffer resource is created and managed separately.
- G-buffer stores:
  - albedo
  - normal
  - depth
- Scene pass is now a geometry pass writing into the G-buffer.
- Fullscreen deferred lighting pass is implemented.
- Deferred resize support is implemented:
  - G-buffer attachments are recreated on framebuffer resize
  - projection aspect now follows the live framebuffer size
- Current lighting is:
  - ambient
  - one directional diffuse light
  - five point lights in the Sponza interior
- Point light visualization is implemented:
  - small tetrahedron markers are drawn at each point light position
- Debug mode is implemented:
  - press P to toggle
  - albedo / normal / depth buffer previews are shown at the upper right


==== next in plan ====

1. alpha mask support
- Sponza curtain / foliage / cutout materials should not render as solid opaque surfaces.
- Use SceneRaw::Material.texture_alpha_mask.
- In geometry pass, discard fragments when alpha mask says transparent.

2. light marker polish
- Current tetrahedron markers are always visible.
- Add a toggle if needed so point light markers can be turned on/off during inspection.
- Optionally make them depth-tested later if occluded markers are preferred.

3. debug view polish
- Optionally add mode labels for albedo / normal / depth.
- Optionally add individual toggles or cycling mode later if needed.

4. directional shadow mapping
- Start with the existing main directional light only.
- Reuse alpha mask logic in the shadow caster path once cutout handling is in.
- Apply the shadow term in deferred lighting after the base path is stable.

5. material expansion
- After alpha mask is stable, consider normal map support.
- Tangent / bitangent data is already uploaded, so the path is prepared.

6. later goal
- Once deferred base is stable:
  - point light shadowing
  - shadowing / SSAO style extensions
  - or voxelization / voxel cone tracing preparation
